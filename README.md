# Intro

Greetings!  
I'm Lara Ayne, the creator of this project.

This repository contains an automated testing challenge developed for the **Accenture QA Automation Engineer Challenge**.  
The project covers **API automation and frontend test automation**, following good practices such as **BDD with Cucumber**, clear test structure, and maintainable code.

The tests are designed to validate critical user flows using automated scenarios, ensuring reliability and readability.

## Stacks

### Frontend
- Cypress
- Cucumber (BDD)

### Backend / API
- Node.js
- JavaScript
